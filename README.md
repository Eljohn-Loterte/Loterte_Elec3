# BIG DATA ANALYSIS LAB 3
This laboratory activity explores the practical application of PySpark DataFrames for structured data processing and analysis. Following a standard ETL (Extract, Transform, Load) pipeline, the activity demonstrates how to efficiently manage datasets in a distributed environment. Initially, sample data is extracted and loaded into a Spark DataFrame, where essential data preprocessing techniques such as column selection, filtering, and cleaning out duplicates or missing values are applied.

Leveraging the expressive, SQL like capabilities of Spark, the activity then progresses to advanced operations where complex queries are executed to extract meaningful insights, such as identifying the top 5 most frequent data categories. Finally, the transformed results are loaded and saved into external storage formats like CSV files. Ultimately, this exercise highlights the optimized execution and ease of use that the PySpark DataFrame API offers for big data manipulation.
